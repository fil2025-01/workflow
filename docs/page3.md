# TypeScript Migration

The client-side logic has been migrated from JavaScript to TypeScript to improve type safety and maintainability.

## Changes
- **Source File:** `script.ts` (contains the source code with type annotations).
- **Compiled File:** `script.js` (generated by TypeScript, included in the Rust binary).
- **Configuration:** `tsconfig.json` manages the compilation settings.
- **Dependencies:** `package.json` tracks `typescript`.

## How to Develop
1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Make Changes:**
   Edit `script.ts` instead of `script.js`.

3. **Build:**
   Run the build script to compile TypeScript to JavaScript:
   ```bash
   npm run build
   ```
   *Note: This overwrites `script.js`.*

4. **Run Server:**
   ```bash
   cargo run
   ```
   The Rust server embeds `script.js` at compile time, so ensure you build the JS before running the server.
